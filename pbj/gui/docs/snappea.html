SnapPea and XTrain

The SnapPea connectivity of XTrain serves several purposes:

  • It computes certain geometric properties of the mapping torus of a pseudo-Anosov map, such as volume and fundamental group.
  • It checks whether the mapping torus of the current map is isometric to a given manifold in the census. Elements of the census are selected by name, e.g., m134.
  • It checks whether the mapping torus of the current map is isometric to a manifold read from a file. In order to compare the mapping tori of two pseudo-Anosov maps, for instance, you can simply load the first map into XTrain, export it to SnapPea (i.e., write a triangulation to a file), then load the next map into XTrain and check for isometry to the file you created before.

Installing SnapPea

In order to call SnapPea from within XTrain, you need to have SnapPeaPython installed. Point your browser to http://www.math.uic.edu/~t3m/ and download the tarball SnapPeaPython.tgz.

  • Open a shell and go to the directory that contains the tarball you just downloaded.
  • Unpack the tarball by typing tar xzf SnapPeaPython.tgz.
  • Type cd SnapPeaPython.
  • Compile SnapPea by typing python setup.py build.
  • Install the Python bindings for SnapPea by typing python setup.py install. On some systems, you may need to have root privileges to install the package (say sudo python setup.py install in this case).
